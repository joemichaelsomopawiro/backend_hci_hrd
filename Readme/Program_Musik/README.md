## üìã Daftar Semua Roles dalam Program Musik

1. **Manager Program** - Manajemen program, generate episodes, assign team, approve special budget
2. **Distribution Manager / Manager Broadcasting** - Manajemen distribusi dan broadcasting
3. **Music Arranger** - Arrange lagu, pilih lagu dan penyanyi
4. **Producer** - Review dan approve song proposal, creative work, QC sound engineer editing
5. **Creative** - Buat script, storyboard, budget, jadwal syuting
6. **Sound Engineer** - Recording dan editing vocal
7. **General Affairs** - Proses permohonan dana
8. **Promosi** - Buat BTS video, foto talent, share ke social media
9. **Produksi** - Syuting, input run sheet, upload hasil syuting
10. **Art & Set Properti** - Manage equipment, approve/reject equipment request
11. **Editor** - Edit video, cek kelengkapan file, report missing files
12. **Design Grafis** - Buat thumbnail YouTube dan BTS
13. **Editor Promosi** - Edit BTS, highlight, iklan episode
14. **Quality Control (QC)** - QC semua konten sebelum broadcasting
15. **Broadcasting** - Upload ke YouTube dan website, schedule playlist
